## guitar-app
Node/Express Application where users can CRUD guitars.

# Version 1.0
Release date 06/08/2025

# Features:

Created Views:
-Home Page
-Create/Update guitars
-Show All guitars
-Login page
-Register page

Created Controllers:
-Guitar controller
-User controller

Created Models:
-Guitar controller
-User controller

Database used: MongoDB
